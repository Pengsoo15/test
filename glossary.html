A
Activation Function
A mathematical function in neural networks that determines whether a neuron should be activated, introducing non-linearity.
Agent
An autonomous AI system that perceives its environment and takes actions to achieve specific goals.
Algorithm
A set of rules or instructions given to an AI system to help it learn and make decisions.
Anomaly Detection
The identification of rare items, events, or observations that differ significantly from the majority of the data.
Artificial General Intelligence (AGI)
Hypothetical AI with human-level intelligence that can understand, learn, and apply knowledge across different domains.
Artificial Neural Network (ANN)
Computing systems inspired by biological neural networks that constitute animal brains.
Attention Mechanism
A technique that allows models to focus on specific parts of input data, crucial in transformers and modern NLP.
Autoencoder
A neural network that learns to compress data into a lower-dimensional representation and then reconstruct it.
B
Backpropagation
An algorithm for training neural networks by adjusting weights based on the error gradient.
Batch Normalization
A technique to improve neural network training by normalizing layer inputs, making training faster and more stable.
Batch Size
The number of training examples used in one iteration of model training.
BERT (Bidirectional Encoder Representations from Transformers)
A transformer-based model designed to understand context in text by looking at words from both directions.
Bias
Systematic errors in AI models that can lead to unfair or inaccurate predictions.
Big Data
Extremely large datasets that can be analyzed computationally to reveal patterns and trends.
C
Chatbot
An AI program designed to simulate conversation with human users through text or voice.
Classification
The task of predicting the category or class of given data points.
Clustering
An unsupervised learning technique that groups similar data points together.
Computer Vision
AI technology that enables computers to interpret and understand visual information from images or videos.
Convolutional Neural Network (CNN)
A type of deep learning network especially effective for processing images and spatial data.
Cross-Validation
A technique for assessing model performance by dividing data into multiple subsets for training and testing.
D
Data Augmentation
Techniques to artificially increase training data by creating modified versions of existing data.
Data Mining
The process of discovering patterns and knowledge from large amounts of data.
Decision Tree
A machine learning model that makes predictions by learning simple decision rules from data features.
Deep Learning
A type of machine learning using multi-layered neural networks to analyze data with complex structures.
Diffusion Model
A generative model that creates data by gradually denoising random noise, used in image generation.
Dimensionality Reduction
Reducing the number of variables in data while retaining important information.
Dropout
A regularization technique that randomly ignores neurons during training to prevent overfitting.
E
Edge Computing
Processing data near the source of data generation rather than in centralized cloud servers.
Embedding
A representation of data (like words) as dense vectors in a continuous space.
Ensemble Learning
Combining multiple models to produce better predictions than any single model.
Epoch
One complete pass through the entire training dataset during model training.
Explainable AI (XAI)
Methods and techniques that make AI decision-making processes transparent and understandable to humans.
F
Feature
An individual measurable property or characteristic of a phenomenon being observed.
Feature Engineering
The process of using domain knowledge to create features that make machine learning algorithms work better.
Few-Shot Learning
The ability of a model to learn from only a few examples of a new task.
Fine-tuning
Adapting a pre-trained model to a specific task by training it further on task-specific data.
Federated Learning
A machine learning approach where models are trained across multiple decentralized devices without sharing raw data.
G
GAN (Generative Adversarial Network)
Two neural networks (generator and discriminator) that compete to create realistic synthetic data.
Generative AI
AI systems that can create new content like text, images, audio, or video based on training data.
GPT (Generative Pre-trained Transformer)
A family of large language models designed to generate human-like text.
Gradient Descent
An optimization algorithm used to minimize the loss function by iteratively adjusting model parameters.
GPU (Graphics Processing Unit)
Specialized hardware that accelerates the training and inference of AI models through parallel processing.
H
Hallucination
When an AI model generates false or nonsensical information that seems plausible.
Hyperparameter
Configuration settings used to control the learning process, set before training begins.
Hyperparameter Tuning
The process of optimizing hyperparameters to improve model performance.
I
Image Recognition
The ability of AI to identify objects, people, places, and actions in images.
Inference
The process of using a trained model to make predictions on new, unseen data.
Interpretability
The degree to which a human can understand the reasoning behind an AI model's decisions.
K
K-Means Clustering
An unsupervised learning algorithm that partitions data into K distinct clusters.
Knowledge Graph
A network of entities and their relationships, used to represent and reason about knowledge.
L
Labeled Data
Training data that has been tagged with correct answers or classifications.
Large Language Model (LLM)
AI models trained on vast amounts of text data to understand and generate human language.
Latency
The time delay between input and output in an AI system.
Learning Rate
A hyperparameter that controls how much model weights are adjusted during training.
LSTM (Long Short-Term Memory)
A type of recurrent neural network capable of learning long-term dependencies in sequential data.
Loss Function
A mathematical function that measures how far a model's predictions are from the actual values.
M
Machine Learning
A subset of AI that enables systems to learn and improve from experience without being explicitly programmed.
Model
A trained AI system that has learned patterns from data and can make predictions.
Model Deployment
The process of making a trained model available for use in production environments.
Multimodal AI
AI systems that can process and understand multiple types of data (text, images, audio) simultaneously.
N
Named Entity Recognition (NER)
NLP task of identifying and classifying named entities (people, places, organizations) in text.
Natural Language Processing (NLP)
AI's ability to understand, interpret, and generate human language.
Neural Network
A computing system inspired by biological neural networks that processes information through interconnected nodes.
Normalization
Scaling data to a standard range to improve model training performance.
O
Object Detection
Computer vision task of identifying and locating objects within images or videos.
One-Hot Encoding
A method of representing categorical variables as binary vectors.
Optimization
The process of adjusting model parameters to minimize the loss function and improve performance.
Overfitting
When a model learns training data too well, including noise, resulting in poor performance on new data.
P
Parameters
Internal variables of a model that are learned from training data (e.g., weights and biases).
Perceptron
The simplest form of a neural network, consisting of a single neuron.
Pooling
A down-sampling operation in CNNs that reduces spatial dimensions while retaining important features.
Pre-training
Training a model on a large general dataset before fine-tuning it for specific tasks.
Precision
The ratio of true positive predictions to all positive predictions made by the model.
Prompt
Input text or instructions given to an AI model to generate desired outputs.
Prompt Engineering
The practice of designing effective prompts to get better outputs from AI models.
Q
Q-Learning
A reinforcement learning algorithm that learns optimal actions by estimating quality values.
Quantization
Reducing the precision of model weights to decrease model size and improve inference speed.
R
Random Forest
An ensemble learning method that combines multiple decision trees for better predictions.
Recall
The ratio of true positive predictions to all actual positive instances in the data.
Recurrent Neural Network (RNN)
Neural networks designed to process sequential data by maintaining memory of previous inputs.
Regression
A prediction task where the model outputs continuous numerical values.
Regularization
Techniques used to prevent overfitting by adding constraints to the model.
Reinforcement Learning
A learning approach where an agent learns by interacting with an environment and receiving rewards or penalties.
ReLU (Rectified Linear Unit)
A popular activation function that outputs the input if positive, otherwise zero.
Retrieval-Augmented Generation (RAG)
A technique that enhances LLMs by retrieving relevant information from external sources before generating responses.
S
Semantic Search
Search technology that understands the meaning and context of queries rather than just matching keywords.
Sentiment Analysis
NLP technique to determine the emotional tone (positive, negative, neutral) of text.
Sigmoid Function
An activation function that maps values to a range between 0 and 1, often used in binary classification.
Speech Recognition
AI technology that converts spoken language into text.
Supervised Learning
Machine learning using labeled training data where the correct outputs are provided.
Support Vector Machine (SVM)
A supervised learning algorithm that finds the optimal boundary between different classes.
Synthetic Data
Artificially generated data used to train models when real data is limited or unavailable.
T
Temperature
A parameter controlling randomness in AI text generation; higher values produce more creative outputs.
Test Set
A portion of data used to evaluate final model performance after training is complete.
Text-to-Speech (TTS)
AI technology that converts written text into spoken audio.
Tokenization
Breaking down text into smaller units (tokens) like words or subwords for processing.
Training Data
The dataset used to teach an AI model patterns and relationships.
Transfer Learning
Applying knowledge learned from one task to help solve a different but related task.
Transformer
A neural network architecture that uses attention mechanisms, powering modern LLMs like GPT and BERT.
U
Underfitting
When a model is too simple to capture the underlying patterns in the data.
Unsupervised Learning
Machine learning using unlabeled data where the model finds patterns without explicit guidance.
V
Validation Set
A portion of data used to evaluate model performance during training and tune hyperparameters.
Vanishing Gradient
A problem in deep networks where gradients become too small, preventing effective learning.
Vector
A mathematical representation of data as an array of numbers used in AI computations.
Vision Transformer (ViT)
Applying transformer architecture to computer vision tasks, treating images as sequences of patches.
W
Weight
Parameters in a neural network that determine the strength of connections between neurons.
Word Embedding
Dense vector representations of words that capture semantic relationships.
Z
Zero-shot Learning
The ability of a model to perform tasks it wasn't explicitly trained for, using only its general knowledge.
